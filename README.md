# Odoo practice
Odoo practice là một project tùy chỉnh một số phân hệ trong chuỗi cung ứng sản xuất ví dụ như: Sales, Purchases, Inventory, Manufacturing, Accounting, etc,... hoặc một số phân hệ đơn lẻ
như addon bm_fleet_gps dựa trên module Fleet.
Dự án dựa trên tiêu chuẩn best practice Odoo 18 Enterprise, ngoài ra còn chứa một số tài liệu hướng dẫn người dùng thao tác các bước sử dụng các module trên hệ thống Odoo Enterprise 18, tài liệu
nghiệp vụ & kỹ thuật của các modules trong chuỗi cung ứng.
